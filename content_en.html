HandyMiner Goldshell - Quick Start Guide
  • Download HandyMiner Release ZIP or Download HandyMiner Release Tarball
  • Prerequisites:
    • Install nodeJS
    • (windows only) Install git bash into Program Files/Git
    • (windows only) Install Virtual COM Port Drivers from Goldshell for USB ASIC support
    • CLI: OR (windows) Double Click install.windows.bat
  • Running:
    • Plug in the power to the Goldshell HS1 and connect the USB to your computer
    • Windows: double click 'dashboard.windows.bat'
    • Linux: Read Linux Notes First and then double click 'dashboard.sh'
    • Mac: double click 'dashboard.mac.command'
    • CLI: or
  • Configuring: First run will start the configurator
    • Set a mode ( solo | pool )
    • Pool Hosts:
      • DXPool port:
        username: registered_username.workerName
        password: anything
      • F2pool port:
        username: wallet.workerName
        password: anything
      • HNSPool port:
        username: registeredUsername or wallet.workerName [unregistered]
        password: registeredUserPassword or anything [unregistered]
  • Wallets:
    • Bob Wallet (GUI, Non-Custodial)
  • Calculators:
    • GPU Mining Calculator
    • ASIC ROI Calculator
