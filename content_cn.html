HandyMiner Goldshell - 快速入门指南
  • 下载 HandyMiner Release ZIP 或下载 HandyMiner Release Tarball
  • 先决条件：
    • 安装 nodeJS
    • （仅限 windows）将 git bash 安装到 Program Files/Git
    • （仅限 windows）将 Virtual COM Port Drivers from Goldshell for USB ASIC support
    • CLI: 或 (Windows) 双击"install.windows.bat"
  • 运行：
    • Windows：双击“dashboard.windows.bat”
    • Linux：Read Linux Notes First 双击“dashboard.sh”
    • Mac：双击“dashboard.mac.command”
    • CLI： 或
  • 配置：首次运行将启动配置器
    • 设置模式（单独 | 库）
    • 库主机：
      • DXPool 端口：
        用户名： registered_username.workerName
        密码： anything
      • F2Pool 端口：
        用户名：wallet.workerName
        密码：anything
      • HNSPool 端口：
        用户名：registeredUsername
        密码：registeredUserPassword
  • 钱夹：
    • Bob Wallet（GUI，非监禁）
  • 计算器：
    • GPU Mining Calculator
    • ASIC ROI Calculator
