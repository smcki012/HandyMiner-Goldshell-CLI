HandyMiner Goldshell - Руководство пользователя
  • Загрузка HandyMiner Релиз ZIP или Загрузка HandyMiner Релиз Tarball
  • Зависимости:
    • Устанавливаем nodeJS
    • (Только для windows) Устанавливаем git bash в директорию Program Files/Git
    • (Только для windows) Устанавливаем Virtual COM Port Drivers from Goldshell for USB ASIC support
    • CLI запуск: или (Windows) Double Click install.windows.bat
  • Запуск:
    • Для Windows: Двойной щелчек по 'dashboard.windows.bat'
    • Для Linux: Read Linux Notes First Двойной щелчек по 'dashboard.sh'
    • Для Mac: double click 'dashboard.mac.command'
    • CLI запуск: или
  • Конфигурация: Первый запуск майнера запросит конфигурацию оборудования
    • Установка режима соло либо пул ( solo | pool )
    • Адреса пулов:
      • DXPool порт:
        Имя пользователя: зарегистрированноеИмяПользователя.имяПользователя
        Пароль: любой
      • F2Pool порт:
        Имя пользователя: Адрес.имяПользователя
        Пароль: любой
      • HNSPool порт:
        Имя пользователя: зарегистрированноеИмяПользователя
        Пароль: Зарегистрированный пароль
